\doxysubsection{Detailed Description}
Structure representing a sprite. 

The \mbox{\hyperlink{struct_sprite}{Sprite}} structure represents a sprite, which is a graphical object with a specific height, width, and color properties. It contains the following members\+:
\begin{DoxyItemize}
\item height\+: Height of the sprite in pixels.
\item width\+: Width of the sprite in pixels.
\item colors\+: Pointer to an array of color values representing the pixels of the sprite.
\item color\+: Color value representing the default color of the sprite.
\item pressed\+: Flag indicating whether the sprite is pressed or not. 
\end{DoxyItemize}
